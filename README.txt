

 Ecomify Website

This is an Ecomify website built using the Spring framework. The website enables users to browse products, add products to their cart, and complete the checkout process using secure payment methods.

 Features

- Browse products by category
- Search for products by keyword
- Add products to cart and manage cart items
- Complete checkout process using secure payment methods
- Register and login as a user to save shipping and payment information
- View order history and order details

 Technologies

- Spring Boot
- Spring MVC
- Spring Security
- Spring Data JPA
- Hibernate
- Thymeleaf
- Bootstrap
- MySQL Database

 Setup

To run the project:

1. Clone the repository
2. Configure MySQL database settings in `application.properties`
3. Build the project using Maven
4. Run the project using the Spring Boot Maven plugin
